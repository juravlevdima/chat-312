# Auth template

## Backend:
* Node.js
* Express.js
* Express Validator
* MongoDB / Mongoose
* Passport.js / JSON Web Token
* Bcrypt.js

## Frontend:
* React (Create React App)
* Redux / ReduxThunk
* TailwindCSS
* Axios

## In the project directory, you can run:

### Installation

```Shell
yarn install && yarn run client:install
```
Installs the dependencies.

### Development

```Shell
yarn run dev
```

### Start

```Shell
yarn run client:build && yarn run start
```
